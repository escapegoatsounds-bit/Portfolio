Visitor Stats

Local traffic log from the editor server.

Heads up: this only counts visits made while editor_server.py was running on this machine (e.g. your own local testing). GitHub Pages is a static host — it can't run this Python backend, so real visitors to the live site are not tracked yet. To capture real traffic you'd need either a small always-on server for this backend, or a privacy-friendly hosted analytics script (e.g. GoatCounter, Plausible) added to the pages instead.
